C3 Mac Setup - Read Me First
=============================

BEFORE YOU START - make sure you have:

  [ ] A Mac running macOS 13 (Ventura) or newer.
  [ ] At least 5 GB of free disk space.
  [ ] You can install software on this Mac (i.e. you know your login
      password and you are an admin user - most people are).
  [ ] A GitHub account. If you do not have one, make one at
      github.com/signup BEFORE running this installer. If you will be
      touching C3 code, ask your manager to add your GitHub username
      to the C3 GitHub organization.
  [ ] Your C3 Microsoft 365 login (the one you use for Outlook/Teams).
  [ ] An Anthropic account at claude.ai (free to make, or sign in
      with your work Google/email).

How to install everything:

  1. Double-click "install.command".
  2. If macOS says "cannot be opened because it is from an unidentified
     developer":
        - Right-click install.command
        - Choose "Open"
        - Click "Open" in the popup
  3. A black Terminal window will open. Follow the prompts.
  4. When asked for a password, type your MAC LOGIN password
     (you won't see characters as you type - that's normal).
  5. Partway through it will ask for your name and email - this just
     labels the code commits you make. Use your work email.
  6. Walk away for 15-30 minutes. Come back when it says "All done!"

After it finishes:

  - Open Warp (Applications folder) and type:  claude
  - Open Claude Desktop (Applications folder) and sign in.
  - First time using Microsoft 365 from Claude, it pops a browser to log in.

If something goes wrong:

  - There is a log on your Desktop named:  claude-install-log.txt
  - Send that file to whoever gave you this installer.
